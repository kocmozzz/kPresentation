kPresentation engine

Created by @kocmozzz

Inspired by Shower presentation engine by Vadim Makeev.

Features

  1. Built on HTML, CSS and vanilla JavaScript, no need in jQuery
  2. Supported browsers: IE 10+, Chrome, Firefox, Opera, Safari
  3. Fully keyboard accessible

  2. Ordered lists begin with a number

    • But do not go too far
    • Otherwise audience will be bored
